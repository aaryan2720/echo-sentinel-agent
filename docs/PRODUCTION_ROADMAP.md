### Current (MVP)
- **Frontend**: React + TypeScript + Vite + shadcn/ui

- **AI**: Hugging Face Inference API
- **Hosting**: Vercel (frontend), Supabase (backend)

### Production (Phase 9)
- **Frontend**: Same (React + TS)
